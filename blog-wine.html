Debian系OSでWineをインストール

2018年3月3日作成, 2019年7月7日更新

結論

以下のコマンドで、64bit版DebianやUbuntuにWineをインストールできます。

sudo dpkg --add-architecture i386
sudo apt update
sudo apt install wine

いきさつ

Debian系のOSにWineをインストールするために「apt install wine」と入力した時、以下のようなエラーが出て失敗しました。

E: 問題を解決することができません。壊れた変更禁止パッケージがあります。

原因を調べたところ、「wine-1.6-i386」という依存パッケージのインストールに失敗している事がわかりました。そしてDebian WikiのWineのページにたどりつきました。

E.g. for amd64 (which most users have) you need i386. Enable it with the following command:

sudo dpkg --add-architecture i386 && sudo apt update

--『Wine - Debian Wiki』より

dpkgに、32bit版のパッケージをインストールする設定が必要だったようです。これで「wine1.6-i386」をインストールでき、Wine自体のインストールもできました。
